# 🎙 Jarvis Voice Assistant (Python)

A simple voice assistant built using Python.

## Features
- Wake word detection ("Jarvis")
- Open Google, YouTube, Facebook, LinkedIn
- Play songs from local music library
- Voice response using pyttsx3

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Webbrowser

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run:
   python jarvis.py
